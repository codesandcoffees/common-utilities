# Common Utilities Package
A package of commonly used utilities

## Available Functions

- Array
  - `toArray(str, delimeter)`
- Boolean
  - `toBool(val)`
- Color
  - `randomPastelColor()`
- Core
  - `isNull(val)`
- Date
- File
- String
  - `toSentenceCase(str)`
  - `toString(arr, delimeter)`
  - `toTitleCase(str)`
