## Available Utilities

- ArrayUtil
  - `strToArray(str, delimeter)`
- BoolUtil
  - `toBool(val)`
- ColorUtil
  - `randomPastelColor()`
- CoreUtil
  - `isExpected(fn, expectedType)`
- DateUtil
- FileUtil
- StringUtil
  - `arrToString(arr, delimeter)`
  - `toSentenceCase(str)`
  - `toTitleCase(str)`
