    public toBool(val: *, trueValuesArr: Array): Boolean source 

    import toBool from 'common-utils-pkg/src/boolean/utils/toBool.js'

    Converts any value to a Boolean equivalent.

    Note: If no user-defined array is passed, the default true values are: ['true', 'yes', 'y', '1'].

    Params:

    NameTypeAttributeDescription
    val *

    value to check if true or false
    trueValuesArr Array
    • optional

    array of user defined true values

    Return:

    Boolean

    Returns a boolean equivalent of the value.

    Example:

    
BoolUtil.toBool('y');
 // => true

BoolUtil.toBool(null);
// => false
