ArrayUtil
strToArray(string, delimiter)

Splits a string to an array based on the delimiter passed.

Arguments
  • string (String) string to convert to array
  • delimiter (Number) where to split the string
Returns

(Array) Returns an array of the split string.

Example 
                ArrayUtil.strToArray('apple;banana;orange', ';');
                // => ['apple', 'banana', 'orange']
BoolUtil
toBool(val, [trueValuesArr])

Converts any value to a Boolean equivalent.

Arguments
  • val (*) value to check if true or false
  • [trueValuesArr] (Array) optional array of user defined true values
Returns

(Boolean) Returns a boolean equivalent of the value.

Example 
                  BoolUtil.toBool('y');
                  // => true

                  BoolUtil.toBool(null);
                  // => false

FileUtil
convertBytes(bytes, decimals)

Converts a number of bytes to a human readable file size.

Arguments
  • bytes (Number) number of bytes to show
  • decimals (Number) number of decimals to show
Returns

(String) Returns a human readable file size.

Example 
                FileUtil.convertBytes(1234);
                // => 1.205078125 kB

                FileUtil.convertBytes(1234, 2);
                // => 1.21 kB
StringUtil
arrToString(array, delimiter)

Convert an array to a string and combine on the delimiter passed.

Arguments
  • array (Array) array to convert to string
  • delimiter (String) characater to use in combining to string
Returns

(String) Returns a string from the array.

Example 
                StringUtil.arrToString(['The', 'quick', 'brown'], '-');
                // => 'The-brown-fox'
