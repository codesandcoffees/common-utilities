How to use?

This package provides multiple utilities that you can import on demand.

Right now, the available utilities are:

ES6

For ES6, you can import a single utility like this

import { ArrayUtil } from 'common-utils-pkg';

Or you may choose to import multiple utilities like so

import { ArrayUtil, BoolUtil } from 'common-utils-pkg';
Browser

You can also use this package directly in your browser by importing the script from UNPKG

<script src="https://unpkg.com/common-utils-pkg"></script>

This adds the package to the window object. You can access the common-utils-pkg variable by

window['common-utils-pkg]

And it would be easier to assign the utilities to a variable like

        <script>
          const ArrayUtil = window['common-utils-pkg'].ArrayUtil;
          const BoolUtil = window['common-utils-pkg'].BoolUtil;
          const ColorUtil = window['common-utils-pkg'].ColorUtil;
          const FileUtil = window['common-utils-pkg'].FileUtil;
          const StringUtil = window['common-utils-pkg'].StringUtil;
          </script>

The usage in the browser is a bit complicated since the package exposes 5 available variables.

Usage

When the utilities are ready for use, you can call the available method like so

        ColorUtil.randomPastelColor()
          // => "hsl(24.380295231593017,27.395530909987613%,86.86101287292395%)"

          ArrayUtil.strToArray('apple;banana;orange', ';');
          // => ['apple', 'banana', 'orange']
