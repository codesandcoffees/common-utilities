strToArray

Splits a string to an array on delimeter.

strToArray
Parameters
string (String) string to convert to array
delimiter (String) where to split the string
Returns
Array: Returns an array of the split string.
Example
ArrayUtil.strToArray('apple;banana;orange', ';');
// => ['apple', 'banana', 'orange']

toBool

Converts any value to a Boolean equivalent.

Note: If no user-defined array is passed, the default true values are: ['true', 'yes', 'y', '1'].

toBool
Parameters
val (any) value to check if true or false
trueValuesArr (Array? = predefinedTrueValues) array of user defined true values
Returns
Boolean: Returns a boolean equivalent of the value.
Example
BoolUtil.toBool('y');
 // => true

BoolUtil.toBool(null);
// => false

randomPastelColor

randomPastelColor

convertBytes

Converts a number of bytes to a human readable file size

convertBytes
Parameters
bytes (Number) number of bytes to show
decimals (Number) number of decimals to show
Returns
String: Returns a human readable file size.
Example
FileUtil.convertBytes(1234);
// => 1.205078125 kB

FileUtil.convertBytes(1234, 2);
// => 1.21 kB
